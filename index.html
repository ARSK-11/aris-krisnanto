What is HTML5?

html stands for hypertext markup langguage, it is easy and fun to learn. html describes the structure of web pages. HTML5 is the fifth and current major version of the html standard.

Why learn HTML5?

It is essential to learn html if you want to build web site, you can't build one if you don't know html because it's one of the prerequisite in learning other language used for web development.
